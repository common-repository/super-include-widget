== Description ==

Super Include Widget plugin allow users to include external file using wordpress widget.

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Go to Appearance > Widgets > Super Include Widget
